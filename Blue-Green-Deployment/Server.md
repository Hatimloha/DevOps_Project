## Server

### Prerequisite
- Name: Server
- EC2: t2.micro

## Update the server 
```bash
sudo -i
apt-get update
apt-get upgrade -y
```

## Change the host name
```
vi etc/host
# remove everthing from the host file and type "Server"
```

## Restart the server
```bash
sudo init 6
```
